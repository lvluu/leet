Introduction

Coding Patterns Documentation

Welcome to the comprehensive guide on common coding patterns used to solve algorithmic problems efficiently.

Note: This repository follows the structure and content from Grokking the Coding Interview by DesignGurus.io, but it's completely free and open-source! 🎉

This documentation covers essential patterns that appear frequently in coding interviews and competitive programming:

Covered Patterns

Two Pointers

The Two Pointers pattern uses indices to traverse data structures efficiently, enabling linear-time solutions for many array and string problems.

Sliding Window

The Sliding Window pattern maintains a range of elements that slides through an array or string to solve problems involving contiguous subarrays or substrings.

How to Use This Guide

Each pattern section includes:

  • When to use: Identifying problem characteristics
  • Common variants: Different approaches within the pattern
  • Pattern recipe: Step-by-step implementation guide
  • Complexity analysis: Time and space considerations
  • Code examples: Multiple language implementations
  • Practice problems: Curated LeetCode problems with direct links

Contributing
